是我小看 firebase studio 了，原来 nix 是可以选择版本的，直接 channel = "unstable" 就可以下载最新的包了。

```
{pkgs}: {
  channel = "unstable";
  packages = [
    # 输入需要安装的包
  ];}
```

我的项目已经完整无碍的在 firebase studio 里运行起来了。firebase studio 的 Gemini 能读到更完整的项目信息，回答的可靠性再次提升，Google 🐮🍺
